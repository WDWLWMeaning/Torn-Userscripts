# Chain Guard

Platform-specific builds for Chain Guard live here.

## Layout

```text
chain-guard/
├── tampermonkey/
│   ├── chain-guard.user.js
│   └── chain-guard.meta.js
├── pda/
│   └── chain-guard.js
└── assets/
```

## Install

- Tampermonkey: `chain-guard/tampermonkey/chain-guard.user.js`
- Torn PDA: `chain-guard/pda/chain-guard.js`

Assets for screenshots or GIFs can go in `chain-guard/assets/`.
