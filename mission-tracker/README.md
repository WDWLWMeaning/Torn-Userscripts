# Mission Tracker

Platform-specific builds for Mission Tracker live here.

## Layout

```text
mission-tracker/
├── tampermonkey/
│   ├── mission-tracker.user.js
│   └── mission-tracker.meta.js
└── assets/
```

## Install

- Tampermonkey: `mission-tracker/tampermonkey/mission-tracker.user.js`

Assets for screenshots or GIFs can go in `mission-tracker/assets/`.
